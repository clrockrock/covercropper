# @covercropper/angular

Angular wrapper for the CoverCropper Web Component.

```bash
npm install @covercropper/angular@beta
```

```ts
import { CoverCropperComponent } from '@covercropper/angular'
```

```html
<cover-cropper-angular src="/cover.jpg" [aspectRatio]="16 / 9" />
```

See the main documentation: https://github.com/clrockrock/covercropper#readme
