# @covercropper/element

The one true CoverCropper Web Component implementation.

```bash
npm install @covercropper/element@beta
```

```ts
import '@covercropper/element'
```

```html
<cover-cropper src="/cover.jpg" aspect-ratio="1.7777778" initial-image-scale="1.4"></cover-cropper>
```

See the main documentation: https://github.com/clrockrock/covercropper#readme
