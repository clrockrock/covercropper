# @covercropper/react

React wrapper for the CoverCropper Web Component.

```bash
npm install @covercropper/react@beta
```

```tsx
import { CoverCropper } from '@covercropper/react'

export function App() {
  return <CoverCropper src="/cover.jpg" aspectRatio={16 / 9} />
}
```

See the main documentation: https://github.com/clrockrock/covercropper#readme
