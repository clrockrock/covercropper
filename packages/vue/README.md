# @covercropper/vue

Vue wrapper for the CoverCropper Web Component.

```bash
npm install @covercropper/vue@beta
```

```vue
<script setup lang="ts">
import { CoverCropper } from '@covercropper/vue'
</script>

<template>
  <CoverCropper src="/cover.jpg" :aspect-ratio="16 / 9" />
</template>
```

See the main documentation: https://github.com/clrockrock/covercropper#readme
